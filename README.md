# Portfolio Website

A modern, high-performance portfolio website for a Data Scientist & AI Engineer, built with Next.js 13, Tailwind CSS, and Framer Motion.

## Features
- **Modern Stack**: Next.js App Router, React 18, TypeScript.
- **Styling**: Tailwind CSS with a custom design system.
- **Animations**: Subtle, professional animations using Framer Motion.
- **Theming**: Dark/Light mode support.
- **Responsive**: Mobile-first design.
- **SEO Optimized**: Metadata and semantic HTML.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

3. **Build for production**:
   ```bash
   npm run build
   ```

## Deployment

The project is optimized for deployment on [Vercel](https://vercel.com).
Simply import the repository and deploy.

## Structure
- `/app`: Main application code (App Router)
  - `/components`: Reusable UI and Section components
  - `/lib`: Utilities and Data layer
- `/public`: Static assets (Images, Resume)

## Customization
Edit `app/lib/data.ts` to update profile information, projects, and skills.
